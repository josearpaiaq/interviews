Find the peaks in a list of numbers

Given a list of numbers, return a list of the peaks in the list.

A peak is defined as a number in the list that is 5 times larger or smaller than its
neighbors.

The first and last numbers in the list are never peaks.

Example 1:

Input: [1,6,2,1]
Output: 1

Example 2:

Input: [0, 10, 0, 10, 0]
Output: 3

Note: 
  - The peaks must evaluate to a number between 0 and 20.
  - The list can have numbers between 0 and 100.
